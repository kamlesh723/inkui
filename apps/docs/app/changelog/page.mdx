# Changelog

All notable changes to InkUI are documented here.

---

## v0.3.0 — April 9, 2026

### Added
- **`npx inkui theme`** — interactive visual theme builder that lets you preview and generate custom themes in the terminal
- **Docs search** — full-text search across all documentation pages
- **Live component previews** — interactive demos embedded directly in component docs pages
- **WebGL hero background** — Hyperspeed-style animated canvas on the landing page
- **"Built with InkUI" showcase** — community projects section on the landing page

### Fixed
- Mobile layout overflow and nav icon alignment
- Sidebar sticky scroll behaviour using `align-self: flex-start`
- Cursor glow intensity on landing page
- Dark theme CSS variables forced at `:root` — fixes invisible text on mobile
- Docs TOC dead space removed

### Changed
- Landing page hero redesigned with terminal background and hover animations
- Footer and landing page nav links synced and updated

---

## v0.2.0 — April 9, 2026

### Added
- **7 new components** — `Toast`, `StatusIndicator`, `LoadingBar`, `Confirm`, `KeyHint`, `Divider`, `Header`
- **`Panel`** — bordered box component with an optional embedded title in the top border
- **`npx inkui playground`** — interactive component browser directly in the terminal
- **`git-tidy`** showcase app — interactive git branch cleanup CLI built with InkUI components
- **`inkui-sysmon`** showcase app — terminal system monitor using InkUI components
- Full light/dark theme support with CSS variables throughout the docs site
- Mobile navigation menu
- `CONTRIBUTING.md`, GitHub issue templates, and PR template

### Fixed
- Critical registry gaps and stale content in the component docs
- Mobile responsiveness and version badge visibility on landing page
- npm auth token wiring for CI publish workflow
- Changeset publish workflow reliability

### Changed
- Landing page logo, favicon, and full visual overhaul
- README updated to reflect all 15 components
- npm scope renamed from `@inkui` to `@inkui-cli`

---

## v0.1.0 — April 7, 2026

### Added
- **Core library** (`@inkui-cli/core`) — design tokens (border styles, spinner frames, spacing) and built-in `darkTheme` / `lightTheme`
- **8 components** — `Spinner`, `Badge`, `ProgressBar`, `TextInput`, `Select`, `MultiSelect`, `Table`, `Dialog`
- Copy-paste installation model via `npx inkui add <component>` — same pattern as shadcn/ui
- TypeScript-first with full type exports
- ESM + CJS dual builds via tsup, compatible with Ink 6 / Node 20+
- Turborepo monorepo with per-package build and test pipelines
- InkUI docs website — Phase 1 (component reference pages, getting started guide)
- Demo GIF and launch-ready README
